vs2019 安装配置MFC

vs2019 MFC

安装MFC

  • 打开vs安装程序

  • 选中c++桌面开发

  • 选择Visual Studio扩展开发

  • （若想快速配置）选择C++ MFC 生成工具（x86 和 x64）进行安装

  • 选择修改，等待安装

  • 创建新项目，选择MFC应用

空项目 配置MFC 快速构建

  • 新建头文件

//应用程序类CWinApp，定义MyAPP类，作为派生类（子类）
class MyApp : public CWinApp
{
public:
//基类的虚函数（接口），子类去实现
//MFC程序的入口地址
virtual BOOL InitInstance();
};//定义类结尾要加;

class MyFrame:public CFrameWnd
{
public:
MyFrame(); //构造函数

};
