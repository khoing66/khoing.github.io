Mysql练习3

  1. 查询男生、女生人数(不重点)

  1. 查询名字中含有”风”字的学生信息（不重点）

  2. 查询1990年出生的学生名单（重点year）

  3. 查询平均成绩大于等于85的所有学生的学号、姓名和平均成绩（不重要）

  4. 查询每门课程的平均成绩，结果按平均成绩升序排序，平均成绩相同时，按课程号降序排列
