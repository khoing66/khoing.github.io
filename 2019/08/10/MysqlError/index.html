MySQL报错error1067

Mysql 报错

widows 无法启动MYSQL服务，error1067

重装

  • C:\Program Files\MySQL
  • 检查C:\WINDOWS目录下是否有my.ini文件,将其删除
  • win+r，弹出运行程序窗口，然后输入
    regedit
    • 删除
      HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\Eventlog\Application\MySQL
  • 这个文件夹，默认是隐藏的，显示隐藏，C:\ProgramData\MySQL Server 5.5
