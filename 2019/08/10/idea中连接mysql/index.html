IDEA连接数据库

IDEA中连接数据库步骤

  1. 打开database
  2. 新建mysql数据库
  3. 勾选与自己电脑安装一样版本的数据库驱动
  4. name 写你自己需要的
    • host 不要改是你的主机名
    • database 写你要连接的数据库 ，不是连接名，不是navicat中的第一层的名称，而是在连接中建的数据库名称，连接之后就能创建表。
  5. 注意在naviicat 中创建数据库时，设置编码格式UTF-8
